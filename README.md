
# Real-Time Unit Traceability System for the EdgeSensor Assembly Process

## 📖 Overview

This system enables **real-time tracking and monitoring of units** throughout the EdgeSensor assembly process at **MKFF LASERTEKNIQUE International Inc.**. It allows production staff and management to track the status, location, and progress of each unit across multiple assembly stations, improving efficiency, accountability, and traceability in the manufacturing workflow.

---

## 🛠️ Technology Stack

### Frontend
- **React.js** – For building interactive user interfaces  
- **Bootstrap 5** – For responsive design  
- **JSX** – For seamless integration of JavaScript and HTML in React components  

### Backend
- **PHP** – For server-side logic and API development  
- **Node.js + Express.js** — For analytics, delay root-cause detection

### Database
- **MySQL** – For secure and structured storage of unit and production data
